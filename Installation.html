InstallationÂ¶

Install via the Slicer Extension Manager of the 3D Slicer Nightly version:

  1. Download, install, and run the 3D Slicer Nightly Build
  2. Select View -> Extension Manager
  3. Hit the INSTALL button for SlicerITKUltrasound
  4. Restart 3D Slicer
  5. The modules will be present in the Ultrasound category
