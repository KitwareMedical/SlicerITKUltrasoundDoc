Installation
============

Install via the `Slicer Extension Manager
<https://www.slicer.org/wiki/Documentation/Nightly/SlicerApplication/ExtensionsManager>`_
via the Nightly version of `3D Slicer <https://www.slicer.org/>`_.

- Download, install, and run the 3D Slicer `Nightly Build
  <http://download.slicer.org/>`_
- Select **View -> Extension Manager**
- Hit the **INSTALL** button for *SlicerITKUltrasound*
- Restart 3D Slicer
- The modules be present in the *Ultrasound* category
